# Unity I2 Localization AI Translator

这是一个单文件 HTML 工具，专为 Unity 多语言插件 **I2 Localization** 导出的 CSV 格式设计。通过接入大语言模型（LLM）API，提供针对游戏文本的批量翻译、语境提取与自动化校对（LQA）工作流。

全程在本地浏览器运行，无需后端环境，确保未公开游戏项目的文本数据安全。

## 🎯 核心解决的问题

I2 Localization 导出的 CSV 包含了 `Key`, `Type`, `Desc` 等元数据列，传统机翻工具无法有效利用这些信息。本工具专门针对此结构优化：
1. **精准的微观语境**：自动提取 `Desc` 列，并将其作为当前行文本的上下文提示（如区分 `UI/Button` 还是 `Dialog`），大幅提升翻译准确度。
2. **富文本标签保护**：强约束大模型在翻译时，完整保留游戏文本中常见的 `<color>` 标签、`{0}` 占位符以及 `\n` 换行符。

## ✨ 主要功能

* **全局语境提炼**：支持“分析全文背景”功能。工具会遍历基准语言全文，让 AI 自动提炼世界观、术语规则与角色语气，并作为后续操作的全局 System Prompt。
* **防截断动态分块 (Dynamic Batching)**：API 请求按“30行 / 1500字符”双条件动态封包，避免因单次请求文本过长导致大模型输出 JSON 被截断。
* **自动化 LQA 与差异审阅 (Diff Review)**：
  * 校对模式下，AI 将针对富文本标签丢失、漏译、语气不符进行排查。
  * 提供独立的“审阅修改”弹窗，展示 `Key`、`Desc`、`原译文 ➔ 新译文` 及 `修改原因`。
  * 引入“脏数据锁定”机制，确认保存后方可导出，且支持单条记录的无损撤销与恢复。
* **断点续传机制**：支持“仅翻译空白”模式，配合任务批次控制，遇到 API 报错或手动停止时，可随时从中断处继续，不浪费 Token。
* **并发控制**：支持在偏好设置中自定义并发请求数（1-5），适配不同等级 API 的并发频率限制（Rate Limit）。

## 🛠️ 快速上手

1. **环境准备**：克隆本仓库或直接下载 `TranslatorTool.html` 文件，使用任意现代浏览器打开。
2. **配置 API**：点击右上角【偏好设置】，填入 `API Key`、`Base URL` 及 `模型名称`。
   > **💡 提示：支持哪些大模型？**
   > 本工具采用标准的 `/chat/completions` API 规范。这意味着你不局限于 ChatGPT，还可以自由接入：
   > * **国产大模型**（如 DeepSeek, Kimi, 智谱, 通义千问等）；
   > * **第三方模型聚合平台**（如 OpenRouter, SiliconFlow 等）；
   > * **本地免费模型**（配合 Ollama 或 LM Studio，Base URL 填入 `http://localhost:11434/v1` 即可实现断网本地翻译）。
3. **导入数据**：将 I2 Localization 导出的 `.csv` 或 `.txt` 文件拖拽至工作区。
4. **语言映射**：在左侧面板选择“基准语言”（原文）和需要处理的“目标语言”。
5. **执行任务**：
   * 点击【开始翻译】填充空白语言列。
   * 点击【开始校对】对已有翻译进行质量审查。
6. **导出回填**：完成审阅后，点击【导出 CSV】，得到的文件可直接导回 Unity I2 Localization 插件中。

## 🔒 隐私与安全性

本工具为纯前端实现。CSV 文件解析、API 请求组装及数据状态管理均在本地浏览器的内存中完成。除直接向您配置的 LLM API 接口发送必要的文本以获取翻译外，不会收集或上传任何数据至第三方服务器。
